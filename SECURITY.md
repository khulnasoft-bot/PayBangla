# Security Policy

## Supported Versions

Only the latest stable version of PayBangla receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.1.0   | :white_check_mark: |
| < 1.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of our users seriously. If you discover a security vulnerability within PayBangla, please let us know immediately.

**Do not open a public issue.** Instead, please send an email to:
📧 **security@khulnasoft.com**

Please include:
*   A description of the vulnerability.
*   Steps to reproduce the issue.
*   Your name (if you wish to be credited).

We will acknowledge your report within 48 hours and provide a fix as soon as possible.
