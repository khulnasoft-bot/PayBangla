# Contributing to PayBangla

Thank you for your interest in contributing to **PayBangla**! As the premier mobile banking gateway for WooCommerce in Bangladesh, we welcome contributions that improve security, speed, and user experience.

## 🚀 How to Contribute

1. **Fork the Repository**: Create your own copy of the project.
2. **Create a Branch**: Use a descriptive name like `feat/new-sms-provider` or `fix/checkout-mobile-css`.
3. **Commit with Emojis**: We follow a clean, emoji-based commit convention.
4. **Write Tests**: If you add new logic, please update `tests/test-paybangla.php`.
5. **Submit a Pull Request**: Provide a clear description of your changes.

## 🛠️ Development Setup

*   Requires PHP 7.4+
*   Requires WooCommerce 5.8+
*   Use `composer install` to set up development dependencies.
